# TheCollector -- FFXIV Dalamud 插件

![TheCollector -- FFXIV Dalamud Plugin Banner](https://ashylila.moe/img/projects/thecollector.png)

> ✨用于自动化收藏品交纳、工票兑换道具自动购买，并打造完美的制作与采集循环的 FFXIV Dalamud 插件。✨

**TheCollector** 是一个用于《最终幻想14》（FFXIV）的 **Dalamud 插件**，通过自动交纳收藏品、在工票达到上限时自动购买工票兑换道具来节省你的时间。
它与 **GatherBuddyReborn** 和 **Artisan** 集成，可配置完整的采集、制作与工票管理自动化循环。

------------------------------------------------------------------------

## ✨ 功能

- 🔁自动交纳收藏品 —— 无需手动点击即可提交物品
- 💰工票达到上限自动购买 —— 不再浪费溢出的工票
- 🛠️与 Artisan 和 GatherBuddyReborn 集成
- ♻️循环自动化 —— 采集、制作、交纳一气呵成

⚠️ 如果你还没有为当前等级解锁对应的 Splendors 兑换页（例如 Lv.80+ 的紫票），插件可能会选错物品。请先与对应的 Splendors 兑换 NPC 对话以解锁。

> ⚠️ **注意**：TheCollector 仍处于早期开发阶段 —— 可能会出现 bug。

------------------------------------------------------------------------

## 📦 安装

1. 在 Dalamud 中打开 **Plugin Settings → Custom Plugin Repositories**
2. 添加以下仓库链接：

   ```txt
   https://raw.githubusercontent.com/Ashylila/AshPluggyRepo/refs/heads/main/repo.json
   ```

3. 在 Dalamud Plugin Installer 中搜索 **TheCollector** 并点击 **Install**
